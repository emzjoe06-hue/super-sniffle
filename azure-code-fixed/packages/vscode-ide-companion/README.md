# Azure Code Companion

The Azure Code Companion extension pairs with
[Azure Code](https://github.com/google-gemini/gemini-cli). This extension is
compatible with both VS Code and VS Code forks.

# Features

- Open Editor File Context: Azure Code gains awareness of the files you have
  open in your editor, providing it with a richer understanding of your
  project's structure and content.

- Selection Context: Azure Code can easily access your cursor's position and
  selected text within the editor, giving it valuable context directly from your
  current work.

- Native Diffing: Seamlessly view, modify, and accept code changes suggested by
  Azure Code directly within the editor.

- Launch Azure Code: Quickly start a new Azure Code session from the Command
  Palette (Cmd+Shift+P or Ctrl+Shift+P) by running the "Azure Code: Run"
  command.

# Requirements

To use this extension, you'll need:

- VS Code version 1.99.0 or newer
- Azure Code (installed separately) running within the integrated terminal
